[TaskCreationHelper run.py usage]

Basic syntax: 
- python3 run.py <subcommand> [config.json path] [args..]

List of subcommands:
- help: Print this documentation.
- full: Do full processes for all solution files, then produce the answer.
- produce: Produce the answer with validation on main AC solution only.
- generate: Generate the template code only, then terminate.
